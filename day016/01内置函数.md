# 内置函数

## 数据类型相关

### 反转

- reversed() 保留原来的列表，产生反转的迭代器，新列表 li.reverse()原列表不保留

### 切片

- slice()

## 字符串相关

- format 调整格式 排版格式， 数字格式
- bytes 字符串转为字节 bytes('你好', encoding='utf-8') 网页，视频都是二进制编码
- bytearray 字节数组 修改字符串，不用生成新的，节省内存
- ord unicode字符转为数字
- chr 数字转为unicode字符
- ascii 只打出ascii
- memoryview 查看内存
- %r repr() 原形毕露
- dict
- set
- frozenset

## 数值计算

- len 长度
- enumerate 列表变字典
- any 遇true则true
- all 遇到false则false
- zip 两个列表拉在一起 元祖对
- filter(函数名, 可迭代对象) 过滤
- map(函数名, 可迭代对象) 映射
- sorted() 排序 生成新列表 key=函数 , reverse=True 反序
