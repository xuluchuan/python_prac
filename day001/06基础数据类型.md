# 基础数据类型

## 基础数据类型

- 数字 int -2^63 ~ 2^63-1 + - * / // % ** 

```
print(100, type(100))
```

- 字符串 str 单引号或双引号里 一般用单引号，防止里面有双引号
- type() 查看数据类型

```
print('100', type('100'))
```

- 字符串相加 是字符串的连接

```
a = 'a'
b = 'b'
c = a + b
print(c)
```

- 字符串可以和数字相乘 重复次数

```
print('坚强' * 10)
```

- 字符串换行后上下使用三个双引号

```
msg = """
这
是
诗
"""
print(msg)
```

- 布尔值：bool True或False

```
a = 3
b = 4
print(a > b)
print(True, type(True))
```
