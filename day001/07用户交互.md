# 用户交互

## input()

```python
name = input('请输入你的名字：')
age = input('请输入你的年龄：')
print(name, age)
print('我的名字是' + name, '我的年龄是' + age)
```

- 1.等待输入
- 2.将你的输入的内容赋值给前面的变量
- 3.数据类型为str
